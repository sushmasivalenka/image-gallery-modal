# Image Gallery Modal

## Description
Image Gallery Modal is a responsive web application built using HTML, CSS, and JavaScript. 
It displays images in a grid layout, and when an image is clicked, it opens in a modal popup 
to show a larger preview without navigating away from the page.

## Features
- Responsive image grid
- Modal popup on image click
- Enlarged image preview
- Close modal option
- Clean and user-friendly UI

## Technologies Used
- HTML
- CSS
- JavaScript
